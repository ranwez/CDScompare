
The documentation for the repository is generated using Doxygen. It generates two types of output : HTML and LaTex.

The LaTeX documentation is available in this folder as 'documentation.pdf'.

To use the HTML documentation, run Doxygen using the following command in this folder:

```
doxygen Doxyfile
```

then navigate to the 'html' subdirectory and open 'index.html' with an internet browser.
